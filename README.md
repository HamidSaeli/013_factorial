# Project 013: Calculate Factorial of a Number

## Description
This project calculates the factorial of a number entered by the user.  
It helps beginners learn about loops, multiplication, and basic math logic.

## Skills Learned
- For loops  
- Working with numbers  
- Variables and multiplication  
- User input  

## How to Run
1. Open `factorial.ipynb` in Jupyter Notebook.
2. Run the cells.
3. Enter a positive integer to see its factorial.
